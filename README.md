# TicTacToe_py
pythonでTicTacToeを作成.

## ファイル構成
├─ main.py       # シミュレーション.  
├─ player_ai.py  # ゲームAI.指定したアルゴリズムで行動を決定.  
├─ statevalue.py # 指定したアルゴリズムで状態価値を決定.  
└─ tictactoe.py  # クラスStateを定義.  

## 実行方法
1. TicTacToe_pyをダウンロード.
1. phtyon実行環境で、main.pyを実行.
1. main.pyを書き換えることで、MiniMaxプレイヤー同士のゲームを見ることができます!
